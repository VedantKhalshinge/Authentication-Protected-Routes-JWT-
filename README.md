# Minimal Authentication System

This project is a complete Authentication & Protected Routes application built using React, Node.js, Express.js, JWT, and bcrypt. It features a minimal, clean, and professional UI.

## Features
- Minimalist, handcrafted UI design
- Secure password hashing using `bcrypt`
- JSON Web Tokens (JWT) for authentication
- `httpOnly`, `SameSite` cookies to securely store JWTs
- Protected React routes (Dashboard & Profile)
- Protected Express API endpoints
- Registration and Login forms with validation
- Loading states and Toast notifications

## Getting Started

### Backend Setup
1. `cd server`
2. `npm install`
3. `npm start` (or `node server.js`)
   - The backend runs on `http://localhost:5000`

### Frontend Setup
1. `cd client`
2. `npm install`
3. `npm run dev`
   - The frontend runs on `http://localhost:5173`

Open your browser to `http://localhost:5173`. You can create an account, log in, and view the protected Dashboard and Profile pages.
